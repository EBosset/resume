# CV / Portfolio (FR)

Ce projet est un portfolio Angular déployé sur GitHub Pages.

## Serveur de développement

```bash
ng serve
```
Ensuite, ouvrez `http://localhost:4200/`. L'appli se recharge automatiquement dès que vous modifiez des fichiers.

## Génération de code

```bash
ng generate component nom-du-composant
```

Consultez toutes les commandes disponibles avec :

```bash
ng generate --help
```

## Build

```bash
ng build
```

Le résultat est placé dans `dist/` avec optimisations prod activées par défaut.

## Tests unitaires

```bash
ng test
```

## Tests end-to-end

```bash
ng e2e
```

Angular CLI n’embarque pas de framework e2e par défaut ; choisissez celui qui vous convient.
